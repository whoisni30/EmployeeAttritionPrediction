# Employee Attrition Prediction using Machine Learning

## Project Overview

This project predicts employee attrition using Machine Learning techniques. The objective is to help HR departments identify employees who are at risk of leaving the organization and understand the key factors influencing attrition. By using predictive analytics, organizations can take proactive steps to improve employee retention and reduce recruitment costs.

---

## Problem Statement

Employee attrition negatively impacts productivity, increases hiring costs, and affects organizational growth. This project aims to build predictive models that classify whether an employee is likely to leave the company based on historical HR data.

---

## Dataset

**Dataset:** IBM HR Analytics Employee Attrition & Performance

- Total Records: 1470
- Total Features: 35
- Target Variable: Attrition

## Key Insights

- Employee attrition rate is approximately 16%.
- Monthly Income, Overtime, Age, Total Working Years, and Job Role are among the most influential factors affecting attrition.
- Machine learning models can effectively identify employees at risk of leaving, supporting proactive HR decision-making.
